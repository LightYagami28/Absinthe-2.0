# Changelog

All notable changes to this project will be documented in this file.

## [2.0.4] - 2012

### Added

- iPad 2,4 support

### Fixed

- Lockdown error -5 workaround for improved device communication

## [2.0.3] - 2012

### Changed

- Updated device detection code
- Improved device communication protocol

## [2.0.2] - 2012

### Added

- Support for iPhone 4 (GSM) 5.1.1b build 9B208

## [2.0.1] - 2012

### Fixed

- Hang condition that could occur in certain situations during jailbreak process

## [2.0] - 2012

### Added

- Initial public release
- iOS 5.1.1 untethered jailbreak support
- Support for multiple iPhone, iPad, and iPod touch models
- Cross-platform GUI (macOS, Windows, Linux)
- Automated backup and restore workflow
